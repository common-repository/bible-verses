Shows random Bible verses as widget or using shortcode.

== Description ==

This plugin will create a widget that will show a random bible verse on every page load. Verses are from dailyverses.net.

Apart from widget you can use shortcode [bible-verse]. It can be used anywhere in the page. It will also show verses.


== Installation ==

1. Upload `Bible Verses` plug-in zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see a new Widgets on your widgets manager.
4. Drag and Drop widget to the position where you want to show bible verses.
5. If you want to show bible verses without widgets in pages or posts, use shortcode [bible-verse].
